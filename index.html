02. O QUE POSSO FAZER POR VOCÊ?

#PROJETO01 - ChatBot

Esse projeto foi implementado com JavaScript, simples de rodar ele gera um QRCode para a empresa ativar. Oferece um suporte personalizado e automatizado, otimizando o contato com os usuários, tornando processos mais ágeis.

#PROJETO02 - IA prevendo atraso de Voo

O projeto "IA Aumentada: Prevendo Voos" utiliza machine learning em Python para prever atrasos de voos, analisando dados de voos dentro e fora da área Schengen. Através da análise exploratória, identifica padrões e tendências que ajudam nas previsões.

#PROJETO03 - Sistema de Cadastro

O sistema de cadastro usa CRUD - Criar (Create), Ler (Read), Atualizar (Update) e Deletar (Delete), um dos conceitos úteis não só em Java, mas em todas as linguagens de programação.
